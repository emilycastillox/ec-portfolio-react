# Emily Castillo Portfolio

A modern React/Next.js portfolio website showcasing my work and skills.

## Tech Stack

- Next.js 15
- React 18
- TypeScript
- Tailwind CSS

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser
